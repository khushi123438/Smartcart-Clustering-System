# 🛒 SmartCart Customer Segmentation System

## 📌 Overview

The SmartCart Customer Segmentation System is an Unsupervised Machine Learning project that groups customers based on their purchasing behavior, spending patterns, and engagement levels.

The goal is to help businesses identify different types of customers and create personalized marketing strategies, improve customer retention, and increase overall business growth through data-driven decision making.

---

## 🎯 Problem Statement

Businesses often use the same marketing strategy for all customers, making it difficult to identify:

- High-value customers
- Loyal customers
- Low-engagement users
- Potential churn customers

This project leverages clustering techniques to discover hidden customer segments using transaction and engagement data.

## 🤖 Clustering Implementation

### K-Means Clustering

Customer segmentation was performed using the K-Means clustering algorithm.

### Steps Performed

- Applied K-Means Clustering
- Used Elbow Method to find optimal clusters
- Evaluated clusters using Silhouette Score
- Assigned cluster labels
- Visualized customer segments
